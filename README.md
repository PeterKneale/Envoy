## Installation

via Package Manager

```Install-Package Envoy```

via .NET CLI: 
    
```dotnet add package Envoy```

## Extras

### Autofac integration

via Package Manager

```Install-Package Envoy.Extras.Autofac```

via .NET CLI: 
    
```dotnet add package Envoy.Extras.Autofac```

via Packet CLI

```paket add Envoy.Extras.Autofac```
