## Installing Envoy

via Package Manager

```Install-Package Envoy```

via .NET CLI: 
    
```dotnet add package Envoy```

## Installing Extras

### Autofac container support: [Envoy.Extras.Autofac](/PeterKneale/Envoy.Extras.Autofac)

via Package Manager

```Install-Package Envoy.Extras.Autofac```

via .NET CLI: 
    
```dotnet add package Envoy.Extras.Autofac```
